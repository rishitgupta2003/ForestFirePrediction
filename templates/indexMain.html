How Does It work

The Prediction's Machine Learning Algorithm is trained with a DATASETS which includes 3 different attributes - Oxygen , Temperature and Humidity.

Dataset used for prediction contains 87 distinct data-rows. Users can choose what kind of model they want for prediction. Each Model has different accuracy. Users Can choose one with higher accuracy.

Data entered by user is then fed to the model to predict the outcome. After prediction a prompt will be generated to check if the predicted value is correct or not. According to user response the entered data is then added to the orignal dataset with the correct output to increase the accuracy of the model.
